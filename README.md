# Repositório de Projetos Front-End

Este repositório contém meuss projetos de desenvolvimento front-end, criados a partir de estudos em cursos de desenvolvimento web e desafios de código encontrados na internet. 

### Projetos inclusos:
- Site de tutoriais de matemática (HTML, CSS); 
- Componente de sumário de resultados (HTML, CSS);
- Componente de preview de um produto (HTML, CSS);
- Componente de leitura de QR Code (HTML, CSS);
- Página de reviews (HTML, CSS);
- Jogo de matemática (HTML, CSS, JS);
- Formulário de sign-up com validação client-side (HTML, CSS, JS);
